# app5.js仕様書
## 起動方法
1,必要なパッケージをインストールする．
```npm install express ejs```
2,サーバーを起動する．
```node app5.js```
3,ブラウザにアクセスする
http://localhost:8080/機能名
## 機能一覧
機能名 | 説明
|---------|---------|
hello1|挨拶を二つ表示
hello2| 挨拶を二つ表示
icon| appleのアイコンを表示
luck|おみくじをして，結果を表示
luck2|ジャンケンの相手の手を決め，表示
luck3|コイントスをして，結果を表示
janken|自分の手と相手の手からジャンケンをして，勝敗，試合数，勝利数を表示
